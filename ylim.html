ylimR Documentation

Convenience function to set the limits of the y axis.

Description

Convenience function to set the limits of the y axis.

Usage

   ylim(...)

Arguments

...

if numeric, will create a continuous scale, if factor or character, will create a discrete scale.

Examples

 ylim(15, 20)
 ylim(c(10, 20))
 ylim("a", "b", "c")
 qplot(mpg, wt, data=mtcars) + ylim(15, 20)
