Description

Annotate a plot

Usage

   annotate(geom, x = NULL, y = NULL, xmin = NULL,
     xmax = NULL, ymin = NULL, ymax = NULL, ...)

Arguments

geom

name of geom to use for annotation

x

x position

y

y position

xmin

xmin position

ymin

ymin position

xmax

xmax position

ymax

ymax position

...

other arguments passed to geom as parameters

Examples

 annotate("text", x = 0, y = 0, label = "title")
